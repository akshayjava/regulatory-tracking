## 2024-05-18 - Managing Focus State with Inline Styles
**Learning:** This application relies heavily on inline styles (`style={{...}}`) rather than external CSS files or utility classes. CSS pseudo-classes like `:focus-visible` cannot be applied natively via inline styles, making it difficult to implement standard keyboard focus rings.
**Action:** When adding keyboard navigation to custom elements using inline styles, manually manage the focus state by attaching `onFocus` and `onBlur` handlers to the element to toggle a React state variable (e.g., `focusedId`), and conditionally render the focus ring (e.g., via `boxShadow: focusedId === id ?
